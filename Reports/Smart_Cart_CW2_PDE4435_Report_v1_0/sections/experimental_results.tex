\subsection{Simulation Setup}
All experiments were conducted in Gazebo Harmonic with ROS~2 Jazzy.
The simulated environment contains a supermarket world with two static
shelf models (at $\pm$1.5\,m), a static orange obstacle box, and a
teleop-controlled person model representing the user. The person is
not an autonomous actor; it is driven via keyboard during each trial.

\subsection{Follow-Me Performance}
\textit{[DRAFT --- Data collection in progress.]}

Preliminary trials indicate the cart successfully maintains the 
target following distance of $1.0 \pm 0.15$\,m during straight-line 
tracking, consistent with the \texttt{FOLLOW\_DIST\_M = 1.0} and 
\texttt{DEAD\_ZONE\_M = 0.15} parameters.
