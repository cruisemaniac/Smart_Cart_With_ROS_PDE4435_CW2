% ─── Abstract ────────────────────────────────────────────────────────────────
\begin{abstract}
This paper presents the design, implementation, and evaluation of an
Autonomous Smart Cart system developed using ROS~2 Jazzy and Gazebo
Harmonic simulation. The system integrates Ultra-Wideband (UWB)
positioning with a 2D LiDAR safety layer to enable a differential-drive
robot to autonomously track a user within a simulated indoor retail
environment. A UWB simulator node generates four-anchor range measurements
from odometry data, a 2D Kalman filter smooths the position estimates,
and a proportional follow-me controller drives the cart to maintain a
1.0\,m following distance. Obstacle safety is enforced by a pass-through
node that scales velocity commands based on LiDAR proximity zones before
they reach Gazebo. Simulation results in Gazebo Harmonic demonstrate
successful follow-me behaviour and layered obstacle avoidance within a
simulated indoor retail environment.
\end{abstract}
